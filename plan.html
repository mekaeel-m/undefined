Team plan

Names

Team members: Mekaeel Malik, Andrew Iammancini, Edwin Warriner, Kaihang Hao

Client name: Hansen Cheng (he is a friend of Mekaeel)

Purpose

We are making an app to showcase Roblox projects and animations. We are gonna store these projects inside a database, then display these with a nice user UI. To showcase the projects he has already worked on to attract new customers.

Data Description

Table 1: Stores his past projects.

Fields:

projectImage: BLOB

projectName: VARCHAR

projectDesc: TEXT

projectLink: TEXT (optional)

Functions Preformed By The App

VIEW (Viewing the projects tiles on the showcase page), 3DView (For the 3d Model allowing the user to interact and view all angles of the design), INSERT (Adding in new projects from an admin page)

Role of PHP and JS

We will use PHP to store and access the 3d model as well as the other project information like images and text. We will also use PHP to upload the data from the admin page to the public site. We will use Javascript for the displaying of the 3d model and also other backend activities.

Roles of Team Members

Andrew: Front-End Design & Layout, Mekaeel: Backend Database, Edwin: 3D Model View Developer, Kaihang: Front-End Design (Contact Page)

The Increments
